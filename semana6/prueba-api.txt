Prueba de la ruta POST /registro

Se envio una peticion POST a http://localhost:3000/registro con:
{ "nombre": "Maria", "mensaje": "Hola comunidad" }

El servidor respondio:
{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor recibio correctamente los datos usando req.body y los devolvio en formato JSON.
